# Flight Booking System (Spring Boot skeleton)

This is a generated starter project for the Flight Booking System backend.

How to run:
1. Import as a Maven project in IntelliJ.
2. Configure application.properties (DB + Mail).
3. Run with: mvn spring-boot:run or run FlightBookingSystemApplication in IDE.
4. Swagger UI: http://localhost:8080/swagger-ui.html
